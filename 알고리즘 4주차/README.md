# 증가하는 수열
 - 역추적
 - 1차원 dp를 통해 길이를 유추 할 수 있다면 pair 쌍으로 저장을 한다면 마지막으로 저장된
    (dp에 저장된 현재까지의 가장 긴 수열)에 마지막 위치(인덱스)까지 저장을 하여 역추적으로 찾아갈 수 있음
 -  단 모든 pair쌍은 (0,1)로 초기화=> 0은 역추적을 끝내기 위한 인덱스 번호/ 1은 모든 수는 자기 자신을 포함한 길이가 1임
